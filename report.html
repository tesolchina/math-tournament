2. Mathematical Analysis / 2. 数学分析

Decomposing the Problem / 问题分解

The schedule has two components:

赛程包含两个组成部分：

  1. Pairings (Latin square L): L[r][i] = j means Ai plays Bj in round r.
  2. 对阵方案（拉丁方L）：L[r][i] = j 表示第r轮Ai对阵Bj。
  3. Color matrix F: F[r][i] = 1 means Ai goes first in round r.
  4. 先后手矩阵F：F[r][i] = 1 表示第r轮Ai执先。

Why Cyclic Shifts Fail for n=10 / 为什么n=10时循环移位不可行

Key discovery: For n=10 (10 ≡ 2 mod 4), cyclic shifts are provably infeasible. The B-player balance leads to 2a = 5, which has no integer solution. Confirmed by ILP and LP relaxation.

关键发现：对于n=10（10 ≡ 2 mod 4），循环移位被证明不可行。B队员平衡条件推导出2a = 5，无整数解。经ILP和LP松弛验证确认。

4. The Winning Approach: CP-SAT / 4. 致胜方案：CP-SAT模型

The key insight was to solve pairings and colors jointly.

关键思路是同时求解对阵方案和先后手分配。

Variable变量Meaning含义Count数量
y[r][i][j]1 if Ai plays Bj in round r第r轮Ai对阵Bj时为11000
f[r][i]1 if Ai goes first in round r第r轮Ai执先时为1100
w[r][i][j]y AND f (linearized)y与f的线性化乘积1000

Why CP-SAT won: Unlike fixing the Latin square first (which can make the color sub-problem infeasible), CP-SAT explores the joint space. Its hybrid SAT + constraint propagation + LP approach efficiently prunes infeasible regions.

CP-SAT取胜的原因：不同于先固定拉丁方再搜索先后手（可能导致子问题无解），CP-SAT在联合空间中搜索。其混合SAT + 约束传播 + LP方法能高效剪枝。

8. Lessons Learned / 8. 经验总结

Mathematical Insights / 数学洞见

  • n=8 works with cyclic shifts because 8 ≡ 0 (mod 4)
  • n=8的循环移位可行，因为8 ≡ 0 (mod 4)
  • For n ≡ 2 (mod 4), cyclic shifts create unavoidable parity conflicts
  • n ≡ 2 (mod 4)时，循环移位产生不可避免的奇偶冲突
  • The constraint matrix is not totally unimodular
  • 约束矩阵不是全幺模矩阵

Computational Insights / 计算洞见

  • Decomposition can hurt: Fixing the Latin square first restricts the solution space
  • 分解可能有害：先固定拉丁方会限制解空间
  • Joint optimization wins: CP-SAT finds compatible pairings + colors simultaneously
  • 联合优化制胜：CP-SAT同时找到兼容的对阵方案和先后手分配
  • Specialized solvers matter: CP-SAT solved in 0.3s what others couldn't in minutes
  • 专用求解器至关重要：CP-SAT用0.3秒解决了其他方法数分钟未能解决的问题
